# Reminisce App Features Summary

## Overview
Reminisce is a dementia monitoring prototype that supports patients and caregivers through daily cognitive check-ins, medication tracking, emergency workflows, and performance monitoring.

## Core Features

### 1. Patient Onboarding
- Register a new patient profile.
- Capture basic details needed for app personalization.
- Configure emergency contacts during setup.

### 2. Dashboard Hub
- Central landing page after login/registration.
- Provides quick access to all main functions.
- Acts as the return point for feature flows.

### 3. Memory Test (Chat-Based)
- Conversational cognitive test with guided questions.
- Supports both text response and voice response pathways.
- Produces test outcomes for tracking patient status.

### 4. Voice Input / Speech Recognition
- Record and transcribe spoken answers for memory tests.
- Submit captured responses back into the test flow.
- Improves accessibility for users who prefer speaking.

### 5. Cognitive Results and Trends
- Shows test score summaries and key indicators.
- Surfaces patterns over time for caregiver review.
- Connects to analytics for deeper insights.

### 6. Medication Management
- Displays daily medication schedule.
- Lets users mark doses as taken.
- Supports adherence monitoring through status updates.

### 7. Caregiver Analytics
- Visualizes cognitive and speech performance trends.
- Highlights strengths, declines, and notable patterns.
- Supports caregiver decision-making and follow-up.

### 8. Emergency Contact Setup
- Stores primary and backup caregiver contact details.
- Includes test-alert behavior during configuration.
- Ensures emergency pathways are preconfigured.

### 9. Emergency Detection and Alerting
- Monitors risk indicators tied to patient behavior/performance.
- Triggers alert workflows when thresholds are met.
- Provides rapid actions (call, SMS, location sharing).

### 10. Notification Scenarios
- Supports reminders and alerts for daily operations.
- Includes medication reminders and emergency notifications.
- Reinforces routine engagement and timely response.

## Typical User Flows
- New patient: Register -> Add emergency contacts -> Use dashboard.
- Daily routine: Dashboard -> Memory test and medications -> Review results.
- Caregiver monitoring: Dashboard -> Analytics -> Interpret trends.
- Emergency response: Detection/trigger -> Alert page -> Contact actions.

## Value for Prototype Demonstration
- Demonstrates end-to-end patient and caregiver journey.
- Combines preventive monitoring with reactive emergency support.
- Shows how data from daily interactions informs caregiver insights.
